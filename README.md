# websnap

take screenshots from static websites

```
Options:
      --version         Show version number                            [boolean]
  -w, --www                                                           [required]
  -o, --output                                                        [required]
  -N, --maxConcurrency                                              [default: 4]
  -P, --basePort                                                 [default: 8080]
  -e, --extension                                              [default: ".png"]
  -m, --media                                               [default: "websnap"]
      --help            Show help                                      [boolean]
```
